# Vision3D ✋🎨

Vision3D is an AI-powered hand gesture recognition system that enables users to control a virtual mouse and draw on a digital canvas using only hand gestures. The project uses Computer Vision and MediaPipe to provide a touch-free human-computer interaction experience.

## Features

- 🖐️ Real-time hand tracking
- 🖱️ Virtual mouse control
- 🎨 Air drawing using hand gestures
- ⚡ Real-time gesture recognition
- 📷 Webcam-based interaction
- 🤖 AI-powered hand landmark detection

## Technologies Used

- Python 3.x
- OpenCV
- MediaPipe
- PyAutoGUI
- NumPy

## Hand Gestures

| Gesture | Action |
|---------|--------|
| Index Finger | Move Cursor |
| Pinch | Click |
| Draw Gesture | Draw on Canvas |
| Open Palm | Stop Drawing |
